# Readme

Servidor básico implementado sobre NestJS.
Incluye conexión con MySQL y login utilizando Passport (JWT + Cookies).

## Instalación

```
npm install
npm run build
```

## Configuración

Duplicar .env.example, nombrar .env y cargar directivas de configuración.

## Ejecución

```
npm start
```

## Hot reload

```
npm run start:dev
```
